## Quick Setup Guide: Cloudflare (Recommended)

### Step 1: Sign Up
1. Go to: https://cloudflare.com
2. Sign up (free)
3. Click "Add a Site"
4. Enter: `propertymarketonline.com`

### Step 2: Add Domain
1. Cloudflare will scan your current DNS
2. Review the records (keep all existing ones)
3. Click "Continue"

### Step 3: Get Nameservers
1. Cloudflare will show you 2 nameservers like:
   - `alice.ns.cloudflare.com`
   - `bob.ns.cloudflare.com`
2. Copy both

### Step 4: Update at Your Current Registrar
1. Go to where you bought the domain
2. Find "Nameservers" or "DNS Settings"
3. Replace with Cloudflare nameservers
4. Save

### Step 5: Add Resend DNS Records
1. Go to Cloudflare ‚Üí DNS ‚Üí Records
2. Click "Add record"
3. Add the 3 TXT records from Resend:
   - `resend._domainkey` (TXT)
   - `send` (TXT)
   - `_dmarc` (TXT)
4. Save each

**Done!** DNS updates in 1-5 minutes!
