3. Run database migrations: `cd backend && node scripts/add-listing-type.js`
4. Install dependencies: `npm install` in both `backend/` and `FRONTEND/property-market/`
5. Start backend: `cd backend && npm run start:dev`
6. Start frontend: `cd FRONTEND/property-market && npm run dev`
